# 色計算 - 基本設計書

## 開発環境など
MyLib、MyLibFileなど自前ライブラリは必要に応じて使用。余裕があればライブラリのメンテも並行で行っていく。

#### 設計環境
必要に応じて以下のツールを使用する
* Gravit Designer
* MyNameList
* Draw IO
* Plant UML
* Visual Studio Code

#### 開発環境
: Visual Studio 2019
:: 最新版
: .Net Framework
:: 4.8.1
: Language
:: C#(WPF)
